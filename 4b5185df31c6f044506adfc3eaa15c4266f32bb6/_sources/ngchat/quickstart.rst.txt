Introduction
============

This ngChat tutorial provides a quick overview on how to access and navigate
SEASALT.AI_'s ngChat_ platform.

By the end of this tutorial, you will be able to:
  * Access ngChat
  * Create a Project
  * Orient yourself in the VSCode Editor

Accessing ngChat
=============

ngChat is hosted by SEASALT.AI and can be accessed using any web browser.

**To Access:**

1. Go to `<https://ngchat.seasalt.ai>`_
